# Music Tracker

## Overview
**Music Tracker** is a web application that allows you to manage and explore your music collection.  
You can add songs with the following details:  
- Artist name  
- Song name  
- Year of release  

The app automatically determines the **decade** in which each song was released and displays all songs with their artist, song name, and decade.

All songs are stored in **local storage**, so they persist even after refreshing the page. Semantic HTML has been used to ensure clean, accessible structure.

## Features
- **Add Songs:** Input artist, song name, and release year.  
- **Automatic Decade Classification:** Songs are categorized by the decade of release.  
- **Filtering:**  
  - Filter by artist  
  - Filter by minimum or maximum year  
- **Grouping:** Group songs by year for easier browsing.
